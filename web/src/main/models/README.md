# Placeholder for 3D Models

This directory contains glTF/GLB models for FediQuest AR experiences.

## Required Models

Place the following models in this directory:

- `tree.glb` - Tree planting quest marker
- `recycle_bin.glb` - Recycling station marker  
- `cleanup_bag.glb` - Cleanup zone marker
- `wildflower.glb` - Wildflower garden marker
- `water_station.glb` - Water conservation point
- `birdhouse.glb` - Wildlife habitat marker

## Model Specifications

- **Format**: glTF 2.0 (.glb binary format preferred)
- **Max Size**: 5MB per model (optimized for mobile)
- **Textures**: Compressed PNG or KTX2
- **Scale**: 1 unit = 1 meter
- **Coordinate System**: Y-up, Z-forward
